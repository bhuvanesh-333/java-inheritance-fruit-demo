# Java Inheritance Fruit Demo

Console application showcasing **multi-level inheritance** in Java using a fruit class hierarchy where child classes inherit and extend parent class methods.

## Class Hierarchy
a (apple: red colour)
↓ extends
b (orange: orange colour)
↓ extends
c (mango: yellow colour)
## Features
- **Multi-level inheritance**: c → b → a
- **Method inheritance**: Child objects call parent methods
- **Polymorphism**: Single `c` object accesses all inherited methods
- Clean class structure demonstrating OOP principles

## How to Run
javac Bhuvi.java
java Bhuvi
**Sample Output:**
red colour
orange colur
yellow colour
## Key Learning Concepts
- Multi-level inheritance chain
- Method inheritance from grandparents
- Single object accessing multiple class methods
- OOP fundamentals in action

## Technologies
- Java 8+
- Object-Oriented Programming (OOP)
